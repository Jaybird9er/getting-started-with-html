Unit 1: Understanding the Web and HTML

The Web: A collection of docs made (mostly) with HTML.
HTML docs are connected by hyperlinks. Users connect
to pages, via their computers, by sending out calls
(requests) in HTTP to servers, which then return those
requests as HTML docs.

HTML: HTML stands for Hyper Text Markup Langauge,
or put simply, it is a formating language. HTML can
markup text, reference media, and link to other pages, like
www.google.com.

Each element in a webpage, consists of
opening and closing tags that bookend the content. Bold
and italics are examples of formating tags. Some tags
have attirbutes that allow for linking to other pages,
or adding images:

Other elements include Whitespace, such as Inline or Block
spacing, that dictate layout. Block spacing forms text into
blocks, leaving defined space around each element.
Inline spacing follows the HTML predefined spacing constructs.
